intcode
=======

**intcode** [**-v**] *code* [*input*]

AoC intcode interpreter. **-v** causes instructions, inputs and outputs
to be printed. *code* and *input* must be files or `-` for standard
input. Standard input is also used if *input* is omitted.

-------
